# DEEP-LEARNING-APPROACHES-TO-PREDICT-AND-CLASSIFY-MALIGNANT-TUMOR-WITH-LESION-SEGMENTATION
Brain Tumor Detection System using Deep Learning and MRI Analysis. Implements CNN-based feature extraction, U-Net lesion segmentation, and multi-class tumor classification (Glioma, Meningioma, Pituitary, No Tumor) to enable accurate, fast, and reliable medical diagnosis.
